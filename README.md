# Rock-Paper-Scissors Game

A modern, interactive Rock-Paper-Scissors game built with Vite, React, and CSS.

## Features
- Interactive UI/UX
- Real-time score tracking
- Responsive design
- Smooth animations

## Technologies Used
- React
- Vite
- Custom CSS

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
